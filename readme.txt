ClientExec Bridge is a plugin that integrates the powerfull ClientExec support and billing software with Wordpress.

== Description ==

The ClientExec Bridge plugin integrates your ClientExec support and billing software into Wordpress providing a seamless and consistent user experience to your customers.

Thanks to the theme inheritance feature, you don't need to style your ClientExec installation anymore, the integration ensures that your ClientExec installation looks and feels like your Wordpress site. 

== Installation ==

1. Upload the `clientexec-bridge` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the ClientExec Bridge settings menu and configure the plugin options.
4. Install the plugin by clicking the Install button. This will create a new page titled ClientExec through which you can access the client portal of ClientExec.

Please visit the [Zingiri](http://forums.zingiri.com "Zingiri Support Forum") for more information and support.
